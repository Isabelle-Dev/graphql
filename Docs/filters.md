### Item

- `name` - Name
- `variation` - variation color
- `diy` - diy status
- `kitcost` - kit cost
- `buy` - buy price in bells
- `sell` - sell price in bells
- `color` - object color codes
- `concept` - HHA Concept
- `tag` - object tag
- `catalog` - For sale or Not for sale
